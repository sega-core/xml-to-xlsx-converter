# ШАГ 1: Установка Node.js
https://nodejs.org/

Скачайте LTS версию (рекомендуется для большинства пользователей)
Нажмите на зеленую кнопку "LTS" с надписью "Recommended for most users"
Перезагрузите компьютер (рекомендуется)


# ШАГ 2: Установка зависимостей
install.bat

# ШАГ 3: Запуск веб приложения
start.bat

# ШАГ 4: Открытие веб приложения
ввести в браузер http://localhost:3002
